# Mascot Assets

Export the Figma mascot as a transparent PNG and place it in this folder.

Suggested first asset:

- `mascot_idle.png`

Good export sizes:

- `512x512` for a lightweight first version
- `1024x1024` if the mascot has detailed linework

Later, we can add more poses:

- `mascot_reminder.png`
- `mascot_celebrate.png`
- `mascot_sleepy.png`
